An interactive educational simulation of protein translation from mRNA sequences. Explore the molecular details of how ribosomes read genetic code and synthesize proteins.

Translation Input Parameters

Enter a sequence of RNA nucleotides to translate into a protein. The sequence will be processed in groups of three nucleotides called codons.
Use RNA nucleotides: A, U, G, C (not T). Sequence should be in triplets (codons).

Animation Controls

Select how fast the translation animation should run. Slower speeds are better for educational purposes.
Begin the mRNA to protein translation animation using the sequence you entered
Pause the currently running translation animation
Resume the paused translation animation
Reset the animation and clear all results to start over

Translation Animation Visualization

This area will show an animated visualization of mRNA translation into protein when you start the animation. The visualization includes colored nucleotides, ribosome movement, and growing protein chain.
